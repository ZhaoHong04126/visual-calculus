# 📐 微積分視覺化筆記 (Visual Calculus Notes)

> **這不僅是一本筆記，而是一場數學的視覺盛宴。**

這是一個專為大學微積分初學者打造的「互動式線上參考手冊」。我們揚棄了傳統講義的靜態限制，結合了嚴謹的數學邏輯（如 $\epsilon-\delta$ 證明）與動態幾何視覺化，幫助你直觀地建立對微積分核心概念的深度理解。

🌐 **[點此進入互動學習空間](https://visual-calculus.vercel.app/)**

---

## ✨ 專案亮點 (Key Features)

*   **🕹️ 互動式幾何探索**：動態展示反函數對稱性、極限逼近過程、漸近線行為以及夾擠定理，讓抽象代數瞬間具象化。
*   **📍 智慧導覽體驗**：
    *   **全域側邊欄**：快速切換不同章節。
    *   **精準章節內容導覽 (NEW!)**：每個章節頂部設有「快速導覽列」，支援磨砂玻璃視覺效果與平滑捲動，精準對位教學重點。
*   **🖋️ 教科書級公式渲染**：採用自訂封裝的 KaTeX 元件，完美解決中文（CJK）衝突，呈現最優雅的 LaTeX 聖經級數學排版。
*   **📱 頂級響應式設計**：無論是 30 吋螢幕還是手機，均能自動調整排版與圖表大小，隨時隨地學習。
*   **🌈 現代化審美排版**：選用 HSL 調和色系，具備舒適的留白與卡片化重點整理，降低長篇閱讀的視覺疲勞。

## 📚 知識地圖 (Course Sitemap)

目前專案涵蓋以下核心章節，內容持續高頻更新中：

### **第零章：微積分簡介 (Introduction)**
*   微分與積分的幾何本質對比
*   學習路徑全地圖探索

### **第一章：函數基礎 (Functions)**
*   **冪函數與多項式 (Power & Polynomial)**：理解各種次方函數的圖形特徵。
*   **超越函數 (Transcendental)**：指數與對數（尤拉數 $e$ 的幾何定義）、三角函數。
*   **反函數機制**：一對一測試、定義域限制與反三角函數的直觀幾何法。

### **第二章：基本公式 (Basic Formulas)**
*   **代數與三角公式**：積化和差、和差化積與倍角公式表。
*   **常用不等式**：三角不等式、算幾不等式 (AM-GM) 與柯西不等式。

### **第三章：極限 (Limits)**
*   極限的直觀與嚴格定義（$\epsilon-\delta$ 證明）
*   極限計演算法則與夾擠定理
*   單側極限、無窮極限與漸近線的嚴格行為。

### **第四章：連續性 (Continuity)**
*   連續點與區間連續的定義
*   常見不連續點類型（跳躍、可移除、無窮不連續）
*   重要定理：中間值定理 (IVT) 之視覺化應用。

### **第五章：導數與微分 (Derivatives)**
*   切線斜率與瞬間變化率的定義
*   基礎微分法則、鏈鎖律與高階導數
*   反函數微分與雙曲函數分析。

### **第六章：微分的應用 (Applications of Derivatives)**
*   極值問題、臨界點與絕對極值。
*   羅爾定理與均值定理 (MVT)。
*   單調性測試、凹凸性與二階導數測試。
*   羅必達法則 (L'Hôpital's rule)。

### **第七章：反導數與積分 (Antiderivatives & Integrals)**
*   不定積分與反導數的概念。
*   定積分的黎曼和與幾何解釋。
*   微積分基本定理 (FTC)。

### **第八章：積分的應用 (Applications of Integrals)**
*   平面區域面積與旋轉體體積。
*   弧長與旋轉曲面積。

### **第九章：公式整理 (Formulas)**
*   全課程公式匯整與最後複習表。


    ```bash
    npm install
    ```
3.  **啟動開發服務器**
    ```bash

    ```
